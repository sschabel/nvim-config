# Neovim Configuration

## Requirements
1. [Tree-Sitter CLI](https://github.com/tree-sitter/tree-sitter)
2. C++ compiler (clang preferred)
3. [Ripgrep](https://github.com/BurntSushi/ripgrep)

## Plugins

1. [Autopairs](https://github.com/windwp/nvim-autopairs) - Adds pairs automatically for code blocks
2. [CMP](https://github.com/hrsh7th/nvim-cmp) - A code completion plugin
3. [DAP](https://github.com/mfussenegger/nvim-dap) - Debug Adapter Protocol
4. [Gruvbox](https://github.com/ellisonleao/gruvbox.nvim) - A Neovim theme
5. [Harpoon](https://github.com/ThePrimeagen/harpoon/tree/harpoon2) - "Getting you where you want with the fewest keystrokes."
6. Language Server Procotol (LSP) - lsp.lua contains all references to LSP servers.
7. [Lualine](https://github.com/nvim-lualine/lualine.nvim) - Customizes the statusline
8. [Gitsigns](https://github.com/lewis6991/gitsigns.nvim) - Integrates some git features
9. [Telescope](https://github.com/nvim-telescope/telescope.nvim) - Fuzzy finder
10. [Toggle Term](https://github.com/akinsho/toggleterm.nvim) - Allows you to toggle a terminal (and customize it)
11. [Tree Sitter](https://github.com/nvim-treesitter/nvim-treesitter) - Adds proper code highlighting to Neovim
12. [NVIM Web Devicons](https://github.com/nvim-tree/nvim-web-devicons) - Adds icons
13. [Oil](https://github.com/stevearc/oil.nvim) - Neovim file explorer
